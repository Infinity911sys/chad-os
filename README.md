# C.H.A.D OS
Cognitive Heuristic Adaptive Dynamics Operating System

C.H.A.D OS is a modular, AI‑centric operating environment focused on:
- context
- continuity
- identity
- adaptive intelligence

This repository contains the full source layout:
- `kernel/` — core runtime and orchestration
- `hal/` — hardware and environment abstraction
- `services/` — identity, memory, context, events
- `intelligence/` — heuristics, prediction, learning
- `continuity/` — identity graph and continuity engine
- `apps/` — system tools and user modules
- `ui/` — interaction and rendering
- `brand/` — identity and design language
- `website/` — public documentation site
- `docs/` — technical documentation
- `tests/` — automated tests
- `tools/` — build and deployment utilities
